# 🧥 DAILY CLOSET
> **"오늘 날씨에 딱 맞게!" 실시간 날씨 API와 내 옷장을 연동하는 똑똑한 패션 매칭 및 OOTD 일기장 앱**

**DAILY CLOSET**은 사용자의 실시간 GPS 위치 정보를 활용해 날씨를 확인하고, 내 옷장에 등록된 실제 옷 중에서 오늘 기온에 가장 잘 어울리는 조합을 콕 집어 추천해 주는 **UX/UI 중심의 패션 아카이빙 플랫폼**입니다.

## ✨ 2. 핵심 기능 소개 (Core Features)

### 🚪 1. 문이 열리며 시작되는 3D 인트로 화면

**[ 핵심 요약 ]**
앱 구동 시 옷장 문이 입체적으로 회전하며 열리는 3D 애니메이션을 구현하여, 사용자로 하여금 '나만의 아카이브 옷장'으로 들어간다는 몰입감을 줍니다.

**[ 상세 설명 ]**
* **Matrix4 3D 변환 기전**: 수학적 행렬 연산을 활용하여, 기존의 정적인 인트로를 넘어 문이 현실감 있게 열리는 3D 연출을 적용했습니다.
* **지연 없는 화면 전환**: 앱이 켜지는 3초 동안 날씨 데이터를 미리 준비하여, 메인 화면 진입 시 로딩 없이 즉시 정보를 확인하도록 최적화했습니다.

---

### ☀️ 2. 날씨 대시보드 & '내 옷으로 추천' 기능

**[ 핵심 요약 ]**
실시간 기온 데이터를 바탕으로 내 옷장에 있는 의류 중 가장 어울리는 조합을 스마트하게 추천해 줍니다.

**[ 상세 설명 ]**
* **실시간 날씨 동기화**: GPS를 활용해 현재 위치의 기온과 날씨 상태를 자동으로 가져옵니다.
* **날씨 맞춤형 테마**: 날씨에 따라 앱 전체 배경색과 글자 색상이 동적으로 변하며, 비가 올 때는 감성적인 빗방울 테마가 활성화됩니다.
* **내 옷 매칭 알고리즘**: **[내 옷장 데이터로 매칭 받기]** 버튼을 누르면, 기온 정보를 계산하여 현재 상황에 딱 맞는 상/하의 옷 조합을 필터링해 보여줍니다.

---

### 🗂 3. 한눈에 보는 디지털 옷장 (Closet)

**[ 핵심 요약 ]**
가지고 있는 모든 의류를 갤러리처럼 정갈하게 정렬하여 관리하고, 새로운 옷을 언제든 간편하게 추가할 수 있습니다.

**[ 상세 설명 ]**
* **직관적인 아카이브**: 모든 소장 의류를 3열 그리드 형태로 보기 좋게 배치하여, 옷장에 무엇이 있는지 한눈에 확인할 수 있습니다.
* **유연한 이미지 등록**: 샘플 의상은 물론, 내가 직접 찍은 사진이나 갤러리의 옷 이미지도 즉시 추가하여 나만의 패션 아카이브를 확장해 나갈 수 있습니다.

---

### 🔍 4. 글로벌 날씨 및 기온 검색 (Search)

**[ 핵심 요약 ]**
내가 있는 곳이 아니더라도 세계 주요 도시의 날씨를 확인하거나, 원하는 기온을 입력하여 가상 코디를 테스트할 수 있습니다.

**[ 상세 설명 ]**
* **도시별 날씨 탐색**: 해외 도시 이름을 입력하면 해당 지역의 현재 기상 상황을 즉시 가져와 코디를 제안합니다.
* **가상 코디 테스트**: 원하는 기온 숫자만 입력해도 해당 기온에 맞는 추천 의상이 나타나, 여행지나 다른 지역의 날씨에 맞는 옷차림을 미리 고민해 볼 수 있습니다.

---

### 📝 5. 사진과 날씨가 함께 저장되는 OOTD 일기장 (OOTD Diary)

**[ 핵심 요약 ]**
오늘 입은 착장을 사진으로 남기고, 당시의 날씨 정보와 함께 메모를 덧붙여 기록하는 나만의 패션 일기장입니다.

**[ 상세 설명 ]**
* **데일리 룩 기록**: 카메라나 갤러리를 통해 오늘 입은 옷 사진을 첨부합니다.
* **데이터 자동 결합**: 사진을 저장하는 순간, **당시의 기온 정보와 날짜가 자동으로 세트로 묶여** 타임라인 형태로 영구 저장됩니다. 언제 어떤 날씨에 어떤 옷을 입었는지 추억과 데이터를 한꺼번에 관리할 수 있습니다.

---

### 👤 6. 프로필 대시보드 & 친구 코디 프리뷰 (My Page)

**[ 핵심 요약 ]**
내 활동 통계를 한눈에 확인하고, 친구들이 기록한 패션 무드를 공유하며 소통할 수 있습니다.

**[ 상세 설명 ]**
* **활동 통계 대시보드**: 내가 소장한 옷의 벌 수와 지금까지 기록한 OOTD 일기 건수를 숫자로 보여주어, 나의 아카이브 성장 과정을 보여줍니다.
* **친구 피드 인터랙션**: 친구들의 프로필 카드를 누르면 감성적인 팝업창이 나타나 친구들의 현재 스타일 무드를 확인할 수 있으며, 연결 버튼을 통해 친구의 계정 피드로 방문하는 인터랙션을 즐길 수 있습니다.

## 🛠 3. 사용된 기술 및 라이브러리 (Tech Stack)

* **언어 및 SDK**: Dart / Flutter SDK (최신 규격인 `Material 3` 디자인 시스템을 준수하여 트렌디하고 세련된 UI를 자랑합니다.)
* **외부 패키지 및 종속성**:
```yaml
  dependencies:
    flutter:
      sdk: flutter
    http: ^1.2.0          # 실시간 날씨 데이터를 인터넷으로 수신하기 위한 통신 모듈
    geolocator: ^11.0.0   # 스마트폰 하드웨어의 GPS 센서 연동용
    image_picker: ^1.1.0  # 카메라 촬영 및 스마트폰 갤러리에서 옷/일기 사진을 업로드하기 위한 모듈
